# R0 evidence (durable)

Checked-in artifacts + the generators that reproduce them, so the R0/R1 recipe
decisions do not depend on machine-local `/data/latent-basemap/track1/` scripts.

## Evidence admission — `status-registry.json`

Scientific consumers must discover artifacts through
`basemap.evidence_status.discover_evidence`, which returns only review-accepted
evidence by default. The append-only registry binds each status transition to
the complete evidence bytes and its owner/reviewer source. Provisional,
invalidated, and diagnostic-only artifacts require an explicit opt-in and cannot
silently enter a certifying decision.

## MiniLM substrate passport — `minilm_passport.json`

Generated by `experiments/minilm_passport.py` (deterministic: seed 42, 201k
sample from the ordered MiniLM-15M shard). Headline: `top2_pc_var ≈ 0.0443`
(fraction of TOTAL variance in the top-2 PCs). The earlier `0.28` figure used a
top-10 singular-value denominator — reproduced on purpose as
`pca.top2_over_top10_denom_bug` and guarded by
`tests/test_minilm_passport.py::test_uses_full_spectrum_not_top10_denominator`.

## A3 recipe rescore — `a3_rescore.json` (regeneration)

Generated by `experiments/score_a3_rescore.py`, which pins all six run
directories explicitly (no globbing) and scores them through the canonical
`basemap.panel_v2.score_panel` (FFR over all anchors; purity over FineWeb-only
anchors via a per-metric mask); full provenance (scorer/coords/data/label/
centroid/anchor hashes, commit, formula version).

**Status:** the scorer is checked in and runnable, but a full regeneration is an
8M-scale panel with `k_frac ≈ 8000`, which is far more expensive than the
200k/2M rungs R1 uses (an 8M pass is minutes of GPU; the 2M golden pass is
~7 s). To keep the GPU on R1's critical path, the fresh regeneration is deferred
and can be produced with:

```
python experiments/score_a3_rescore.py --out experiments/evidence/a3_rescore.json
```

The prior machine-local result (corrected FFR, `code_rev 6a304ae`) is preserved
under `legacy/` as the decision-of-record until then: **nomn leads mn4 on FFR
(≈0.563 vs 0.503) and purity across 3 seeds** — the recipe R1 rebaselines under
the `umap` kernel.
